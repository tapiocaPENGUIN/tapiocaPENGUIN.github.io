What is Velociraptor Football?
A JavaScript web app that runs in the browser using pixiJS, which allows you to create and run your own football play
Give it a try here or click the tutorial tab.

Visit our GitHub Repository
Click here to find the source code and README on GitHub.
Velociraptor Football is free to use and you can change it for your needs.

Dependencies
Velociraptor Football uses PixiJS to create the in browser animations.
